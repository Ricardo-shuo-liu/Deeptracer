对于Deeptracer的部分开发要求：  

	1，所有编码以及文件必须使用utf-8
	
	2，所有文件的open必须显式标注encoding='utf-8'
	
	3，有一个issue用于汇报每周进展
	
	4，如果有新的功能建议以及开发问题提交新的issue  

	5，提交pr时候说明清楚增加或实现了了什么功能

	6,python统一使用3.10.19 开发
	
	7，为了保证高效性质函数定义要依照下面格式  
```
def function(element:int)->int:
	"返回1(说明函数的作用！)"
	return 1
```
